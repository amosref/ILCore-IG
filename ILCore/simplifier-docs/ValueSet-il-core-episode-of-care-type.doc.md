# IL Core EpisodeOfCare Type ValueSet
EpisodeOfCare category value set used for IL Core EpisodeOfCare category slices.

## Canonical
[http://fhir.health.gov.il/ValueSet/il-core-episode-of-care-type](http://fhir.health.gov.il/ValueSet/il-core-episode-of-care-type)

## See also
- [IL Core EpisodeOfCare Type CodeSystem](./CodeSystem-il-core-episode-of-care-type.doc.md)

## Includes
- All codes from [http://hl7.org/fhir/ValueSet/episodeofcare-type](http://hl7.org/fhir/ValueSet/episodeofcare-type)
- All codes from [http://fhir.health.gov.il/cs/il-core-episode-of-care-type](http://fhir.health.gov.il/cs/il-core-episode-of-care-type)

## Usage
- Used by IL Core EpisodeOfCare category slices (for example `type[recurring-daily-hospitalization]`).
- For the full `EpisodeOfCare.type` binding, see the union value set [http://fhir.health.gov.il/ValueSet/il-core-episode-of-care-type-and-service-type](http://fhir.health.gov.il/ValueSet/il-core-episode-of-care-type-and-service-type).
