- All codes from [http://fhir.health.gov.il/cs/il-core-episode-of-care-type](http://fhir.health.gov.il/cs/il-core-episode-of-care-type)

## Usage
- Bound to `ILCoreEpisodeOfCare.type` (extensible).
- Enables `EpisodeOfCare.type` to carry both category codes and service type context.
